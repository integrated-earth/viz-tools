# AdvancedSceneExplorer

## setup

1. get nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
or see https://github.com/nvm-sh/nvm

2. get node
```
nvm install --lts
```

3. install packages:
```
$ npm install
```

## testing:
```
$ npm run start
```
	
## release:
```
$ npm run build
```
